## 🚀 How to Run

1. Make sure you have a C++ compiler installed (e.g. [MinGW](https://www.mingw-w64.org/) on Windows or `g++` on Linux/Mac).
2. Clone the repository:
   ```bash
   git clone https://github.com/Arpitwasnotavaliable/C-Daily-Update.git
   cd C-Daily-Update
   ```
3. Compile and run any file:
   ```bash
   g++ forloop.cpp -o forloop
   ./forloop
   ```
